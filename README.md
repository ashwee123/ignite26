# ignite26
Flow State is a productivity-focused web application designed specifically for individuals who feel easily overwhelmed or distracted—especially neurodivergent users navigating the challenges of school, work, and everyday life.

Our goal was to create a supportive, accessible space with three core features:
• Focus – tools to help users stay on task and minimize distractions
• De-stress – resources to manage anxiety and reset mentally
• Advice – guidance and tips for handling real-life challenges
 -With the added benefit of choosing different color palettes based on mood, preference, or color blindness.
